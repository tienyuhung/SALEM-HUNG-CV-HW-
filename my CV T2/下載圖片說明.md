# 如何下載並設置首頁背景圖片

## 方法一：手動下載（最簡單，推薦）

1. **打開 Google Drive 連結**
   - 訪問：https://drive.google.com/drive/u/0/folders/1oK3zLzuQIrhxEG2-z7OnSEqpnaAAp3Bx
   - 找到 `salemhung2025.jpg` 文件

2. **下載圖片**
   - 右鍵點擊 `salemhung2025.jpg`
   - 選擇「下載」
   - 下載完成後，將圖片重新命名為 `hero-image.jpg`

3. **放置圖片**
   - 將 `hero-image.jpg` 移動到專案根目錄（與 `index.html` 同一資料夾）
   - 路徑：`my CV T2/hero-image.jpg`

## 方法二：使用 Python 腳本下載

### 步驟 1：獲取文件的直接連結

1. 在 Google Drive 中右鍵點擊 `salemhung2025.jpg`
2. 選擇「取得連結」
3. 確保權限設為「任何擁有連結的人都可以查看」
4. 複製連結（格式類似：`https://drive.google.com/file/d/1ABC.../view`）

### 步驟 2：執行下載腳本

**選項 A：使用完整連結**
```bash
cd "my CV T2"
python3 download_image.py "https://drive.google.com/file/d/YOUR_FILE_ID/view"
```

**選項 B：只使用文件 ID**
```bash
cd "my CV T2"
python3 download_image.py YOUR_FILE_ID
```

## 方法三：使用瀏覽器直接下載

1. 在 Google Drive 中右鍵點擊 `salemhung2025.jpg`
2. 選擇「取得連結」，確保設為「任何擁有連結的人都可以查看」
3. 從連結中複製文件 ID（`/d/FILE_ID/` 中間的部分）
4. 在瀏覽器中打開以下連結（將 `FILE_ID` 替換為實際 ID）：
   ```
   https://drive.google.com/uc?export=download&id=FILE_ID
   ```
5. 下載後重新命名為 `hero-image.jpg` 並放在專案目錄

---

## 驗證

下載完成後，確認以下檔案存在：
- ✅ `my CV T2/hero-image.jpg`

然後在瀏覽器中打開 `index.html` 即可看到效果。

**注意**：圖片會自動應用以下效果：
- 轉為黑白
- 對比度提升至 180%
- 亮度調整至 85%
- 添加半透明黑色遮罩層
- 所有白色文字都有陰影效果，確保清晰可見
