# Netlify 使用详细步骤

## 为什么选择 Netlify？

- ✅ **最简单**：拖放文件夹即可，无需 Git
- ✅ **自动处理大文件**：不用担心文件大小限制
- ✅ **快速部署**：几秒钟即可完成
- ✅ **免费 HTTPS**：自动提供安全链接
- ✅ **自动更新**：每次上传都会自动更新网站

---

## 第一步：注册/登录 Netlify

1. **访问 Netlify 网站**
   - 网址：https://www.netlify.com
   - 点击右上角的 "Sign up" 或 "Log in"

2. **选择登录方式**
   - **推荐**：点击 "Sign up with GitHub"（如果您有 GitHub 账号）
   - 或者使用邮箱注册：
     - 输入邮箱和密码
     - 点击 "Sign up"
     - 检查邮箱并验证账号

3. **完成注册**
   - 如果使用 GitHub 登录，会跳转到 GitHub 授权页面
   - 点击 "Authorize Netlify"
   - 返回 Netlify，应该已经登录成功

---

## 第二步：部署网站（拖放方式）

### 方法 A：直接拖放文件夹（最简单）⭐

1. **登录 Netlify 控制台**
   - 登录后，您会看到 Netlify 控制台（Dashboard）

2. **找到部署区域**
   - 在控制台页面，找到 "Sites" 区域
   - 您会看到一个虚线框，上面写着：
     ```
     Want to deploy a new site without connecting to Git?
     Drag and drop your site output folder here
     ```
   - 或者点击 "Add new site" → "Deploy manually"

3. **准备项目文件夹**
   - 打开 Finder
   - 导航到项目文件夹：
     ```
     /Users/salemhung/Library/Mobile Documents/com~apple~CloudDocs/Desktop/台科碩班研究所/Cursor HW/my CV T2
     ```

4. **拖放文件夹**
   - 将整个 `my CV T2` 文件夹拖放到 Netlify 的虚线框中
   - 或者点击虚线框，然后在文件选择器中选择项目文件夹

5. **等待部署**
   - Netlify 会自动开始上传和部署
   - 底部会显示上传进度
   - 通常需要 10-30 秒

6. **部署完成**
   - 部署完成后，Netlify 会自动生成一个链接
   - 链接格式：`https://random-name-123456.netlify.app`
   - 您的网站已经可以访问了！

---

## 第三步：自定义网站名称（可选）

1. **进入网站设置**
   - 在 Netlify 控制台，点击您的网站名称
   - 或者点击网站卡片上的 "Site settings"

2. **更改网站名称**
   - 点击 "Change site name"
   - 输入您想要的名称（例如：`salem-hung-portfolio`）
   - 点击 "Save"
   - 新的链接格式：`https://salem-hung-portfolio.netlify.app`

3. **检查可用性**
   - Netlify 会检查名称是否可用
   - 如果已被使用，需要选择其他名称

---

## 第四步：访问和分享网站

1. **获取网站链接**
   - 在 Netlify 控制台的网站卡片上，可以看到网站链接
   - 点击链接即可在浏览器中打开网站

2. **分享链接**
   - 复制网站链接
   - 可以直接分享给其他人
   - 链接格式：`https://your-site-name.netlify.app`

3. **测试网站**
   - 在浏览器中打开链接
   - 检查所有功能是否正常：
     - 首页背景图显示
     - 导航菜单工作正常
     - Portfolio 图片可以点击
     - About 页面显示正常
     - Services 箭头动画正常

---

## 第五步：更新网站内容

### 如果以后需要更新网站：

1. **修改本地文件**
   - 在项目文件夹中修改文件（例如：编辑 `index.html`）

2. **重新部署**
   - 打开 Netlify 控制台
   - 将修改后的文件夹再次拖放到部署区域
   - 或者点击网站卡片 → "Deploys" → "Deploy site" → "Deploy manually"
   - 选择修改后的文件夹

3. **等待更新**
   - Netlify 会自动更新网站
   - 通常几秒钟即可完成

---

## 方法 B：使用 Netlify Drop（更简单）

1. **访问 Netlify Drop**
   - 网址：https://app.netlify.com/drop
   - 或者直接在 Netlify 控制台找到拖放区域

2. **拖放文件夹**
   - 将项目文件夹拖放到页面中
   - 等待部署完成

3. **获得链接**
   - 部署完成后立即获得链接

## 常见问题

### Q1: 上传很慢？
**解决**：
- 这是正常的，因为图片文件较大
- 请耐心等待，通常 10-30 秒即可完成
- 可以在底部查看上传进度

### Q2: 部署失败？
**解决**：
- 检查网络连接
- 确认所有文件都在文件夹中
- 确认 `index.html` 在根目录
- 重新尝试拖放

### Q3: 网站显示 404？
**解决**：
- 确认 `index.html` 在项目根目录
- 检查文件名是否正确（必须是 `index.html`）
- 等待几分钟后重试

### Q4: 如何删除网站？
**解决**：
- 在网站设置中，找到 "General" → "Delete site"
- 输入网站名称确认删除

### Q5: 可以绑定自定义域名吗？
**解决**：
- 可以！在网站设置 → "Domain settings" → "Add custom domain"
- 按照提示配置 DNS 设置

## 重要提示

1. **文件结构**：
   - 确保 `index.html` 在项目根目录
   - 所有文件路径使用相对路径（如 `hero-image.jpg`）

2. **文件大小**：
   - Netlify 对单个文件限制是 100MB
   - 您的文件都在限制范围内

3. **免费额度**：
   - Netlify 免费版每月有 100GB 带宽
   - 对于个人网站完全足够

4. **自动 HTTPS**：
   - Netlify 自动为所有网站提供 HTTPS
   - 链接是安全的（https://）
